- How A2A differs is that it allows multi-turn-interaction between varies domain specific agents.
- Other benefitis of A2A being :
	- It creates a standard interface of communication between agents which in turn
		- reduces integration costs, 
		- improves interoperability
		- robusts application of standard security practices.
	- It also allows Long running asynchronous execution of tasks
	
- Tools made as a wrapper over models restrict their capabilities to bring in some determinism in the behavior a tool is expected of. Hence this is not beneficial in A2A communication.
